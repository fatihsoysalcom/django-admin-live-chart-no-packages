# Django Admin Live Chart No Packages

This example demonstrates adding live, dynamic charts to a web page, simulating the approach for Django Admin. It uses a basic Python server to provide data and client-side JavaScript with Chart.js (via CDN) for visualization. The chart updates automatically every few seconds without requiring any extra Python packages.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run `python main.py` in your terminal.
3. Open your web browser and navigate to `http://localhost:8000`.
